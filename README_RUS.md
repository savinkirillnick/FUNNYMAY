FUNNYMAY API v1.0 (2019-07-05)
=====================
Основная информация API
=====================

* Основная ссылка доступа: https://www.funnymay.com/api/
* Ответы всех ссылок возвращаются в формате JSON.
* Каждая ссылка может вернуться ОШИБКОЙ; при ошибке выводится:

```
{
  "status":0
}
```

ОГРАНИЧЕНИЯ
=====================
Информация о криптовалютных парах обновляется каждые 60 секунд.


Публичный API
=====================

API СИГНАЛОВ
-----------------------------------

### ЗАПРОС
```
GET http://www.funnymay.com/api/signal_api.php
```

### ПАРАМЕТРЫ
Параметр  |Тип      |Обязательный |Описание
----------|---------|-------------|--------------------
s         |СТРОКА   |ДА           |Тип сигнала
p         |СТРОКА   |НЕТ          |Дополнительный параметр

#### Типы сигналов:
Значение        |Описание
----------------|----------------------
gainer          |Новые гейнеры
loser           |Новые лозеры
newhigh         |Новые хаи
newlow          |Новые лоу
sma20xsma50     |Пересечения SMA20 и SMA50
sma50xsma200    |Пересечения SMA50 и SMA200

### ПРИМЕР ЗАПРОСА
```
http://www.funnymay.com/api/signal_api.php?s=gainer
```

### ПРИМЕР ОТВЕТА
```
{
  "status":1,
  "data": [
    {
      "signal":"Gainer",
      "symbol":"RCNBTC",
      "price":0.00123456,
      "change":4.123
    },
    { ... }
  ]
}
```

API СКРИНЕРА
-----------------------------------

### ЗАПРОС
```
GET http://www.funnymay.com/api/screener_api.php
```
### ПАРАМЕТРЫ
Параметр  |Тип    |Обязательный |Описание
----------|-------|-------------|--------------------
bs        |ENUM   |НЕТ          |Базовая валюта
qt        |ENUM   |НЕТ          |Котируемая валюта
vr        |ENUM   |НЕТ          |Относительный объем
ch        |ENUM   |НЕТ          |Изменение, %
qp        |ENUM   |НЕТ          |Эквивалент в USDT
p10       |ENUM   |НЕТ          |Настройки SMA 10 дней
p20       |ENUM   |НЕТ          |Настройки SMA 20 дней
p50       |ENUM   |НЕТ          |Настройки SMA 50 дней
p100      |ENUM   |НЕТ          |Настройки SMA 100 дней
p200      |ENUM   |НЕТ          |Настройки SMA 200 дней
hl50d     |ENUM   |НЕТ          |Настройки 50-дневного Хай-Лоу
hl52w     |ENUM   |НЕТ          |Настройки 52-недельного Хай-Лоу
m         |ENUM   |НЕТ          |Настройки коэффициента Майера
d         |ENUM   |НЕТ          |Количество дней подряд в одну сторону

#### Применимые значения параметров
Все настройки параметров скринера вы можете найти в адресной строке после выбора необходимых фильтров на http://www.funnymay.com/screener.php
